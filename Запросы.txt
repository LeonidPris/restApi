====1. Новая категория: POST ====

http://localhost:8081/category/new

тело:
{
    "categoryName": "Computer accessories"
}
----------------------------------------------------------------------------------------------
====2. Новый товар: POST====

http://localhost:8081/good/new

тело:
{
    "title":"GeForce RTX 5070 future",
    "price":"10000000",
    "quantityInStorage": 1,
    "category":"Computer accessories"
}
----------------------------------------------------------------------------------------------
====3. Добавить количество товара: GET ====

http://localhost:8081/good/add=2&id=1
----------------------------------------------------------------------------------------------
====4. Добавить количество товара: POST. приходит id и количество ====

http://localhost:8081/good/addQuant

тело:
{
    "id":"1",
    "count":"3"
}
----------------------------------------------------------------------------------------------
====5. Добавить количество товара: POST. приходит объект целиком====

http://localhost:8081/good/new

тело:
{
    "id": 1,
    "title":"GeForce RTX 5070 future",
    "price":"10000000",
    "quantityInStorage": 6,
        "category": {
            "id": 1,
            "categoryName": "Computer accessories",
            "goods": []
        }
    }
----------------------------------------------------------------------------------------------
====6. Вывести всю категорию: GET ====

http://localhost:8081/good/category=1
----------------------------------------------------------------------------------------------
====7. Вывести все товары: GET ====

http://localhost:8081/good
----------------------------------------------------------------------------------------------
====8. Отгрузка: работает только после авторизации .POST.====

http://localhost:8081/good/delQuant

тело:
{
    "id":"1",
    "count":"1"
}
----------------------------------------------------------------------------------------------
====8. Создание нового Юзера + создаются Cookie авторизации====

http://localhost:8081/user/create

тело:
{
    "login" : "login1",
    "password" : "789"
}
